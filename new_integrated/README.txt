AE700_Integrated — F-4 Phantom Project
IIT Bombay | Guidance and Control of Unmanned Autonomous Vehicles
================================================================

AIRCRAFT: F-4 Phantom (subsonic cruise configuration)
SOURCE:   Roskam, Airplane Flight Dynamics, Part I, pp. 536-538
UNITS:    ft, slug, lb, rad throughout
REF:      Beard & McLain, "Small Unmanned Aircraft" (2012)


QUICK START
-----------
1. Open MATLAB and add this folder to path:
      addpath(genpath('AE700_Integrated'))

2. Run all sections at once:
      >> run_all

3. Or run individual sections:
      >> Section1_Animation
      >> Section2_EOM
      >> Section3_Forces
      >> Section4_Linear
      >> Section5_Autopilot

4. Build Simulink models:
      >> build_chap2_simulink    % static pose animation
      >> build_chap3_simulink    % 6-DOF EOM driven by force/moment inputs

KEY DESIGN DECISIONS
--------------------
1. Single parameter file (F4_params.m)
   - Replaces F4_Phantom_params.m (workspace scalars) and F4_chap4_params.m (struct P)
   - Both interfaces emitted so Simulink S-Functions and script code work identically

2. Nonlinear CL(alpha) with stall blending (eq 4.9)
   - Sigmoid blending between linear region and flat-plate stall
   - CD(alpha) quadratic polar model (eq 4.11)
   - Linear Cm(alpha) for pitching moment

3. eom.m replaces duplicated local eom functions
   - All Section scripts call eom(x, delta, wind, P) via ode45

4. forces_moments.m — fixes vs previous versions
   - beta clamped to [-1,1] before asin to prevent domain error in gusts
   - Consistent parameter names: P.Cmo (not P.Cm0)

5. compute_trim.m — multi-start Nelder-Mead
   - Searches from 5 starting points; picks best convergence
   - Reports residual J so quality is visible

6. Autopilot gains (Section5)
   - wn_phi=1.0 rad/s  (Dutch roll wn≈5.55 => separation factor ~5.5)
   - wn_h=0.08 rad/s   (phugoid T≈140s => altitude loop much faster)
   - Rudder: P-only (no integrator) avoids Dutch-roll wind-up
   - Va/pitch: negative sign (pitch DOWN increases speed)

7. drawAircraft.m — tail geometry
   - Horizontal tail attached at fuselage mid-ring (no gap)
   - Vertical tail scaled with fuselage cross-section
