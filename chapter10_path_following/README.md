# Chapter 10 — Path Following

This chapter implements the two fundamental path-following primitives described in Beard & McLain Chapter 10: straight-line segment following (Algorithm 5, Eqs 10.1–10.8) and circular orbit following (Algorithm 6, Eqs 10.9–10.15). Both algorithms output autopilot commands — course angle command, roll angle command, altitude command, and airspeed command — which feed into the inner-loop autopilot designed in Chapter 6. The straight-line follower drives the cross-track error to zero via a nonlinear course-angle law (Eq 10.4), while the orbit follower regulates the radial distance error and commands a bank angle consistent with a coordinated turn at the orbit radius (Eq 10.15).

The main deliverable is `path_follower.m`, which dispatches on `path.flag` (1 = line, 2 = orbit). Two tuning parameters must be added to `F4_params.m`: `chi_inf` (the maximum course correction angle for straight-line following, roughly 70 deg) and `k_path` / `k_orbit` (the proportional gains for cross-track and radial-error control). The test script `Section10_PathFollowing.m` tests both modes in a 40-second simulation with a constant lateral wind of [5, -3, 0] ft/s and plots the ground tracks against the reference path.
